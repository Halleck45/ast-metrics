Metric Guide

Maintainability Index (MI)

0–128. Higher is better. Aggregates complexity, size, and churn. Aim ≥ 85.

Cyclomatic complexity

Number of independent paths. ≤ 10 is usually OK. Keep low to ease testing.

LCOM4

Class cohesion. 1 means cohesive. Higher implies split responsibilities.

LLOC

Logical lines of code. Measures size. Prefer smaller units.

Instability

0–1. Outgoing deps ratio. Close to 0 for stable modules, close to 1 for volatile.

Halstead Bugs

Theoretical bugs density from Halstead volume. Use as a relative indicator.
